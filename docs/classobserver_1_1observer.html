Detailed Description

observer class that can be initialized to be run with the cotask scheduler.

See run for behavior.

init is the observer object initializer which takes 6 arguments. The arguments are data shares.

Parameters
left_pos: left_pos is a float share that holds the current position of the left encoder. Used as an input.
right_pos: right_pos is a float share that holds the current position of the right encoder. Used as an input.
yaw: yaw is a float share that holds the IMU reading of the yaw.
yaw_velocity: yaw_velocity is the yaw_velocity as measured by the IMU task.
left_voltage: left_voltage is the PWM value of the left motor which observer scales to the expected voltage.
right_voltage: right_voltage is the PWM value of the right motor which observer scales to the expected voltage.
left_vel: left_vel is the velocity of the left motor.
right_vel: right_vel is the velocity of the right motor.
total_dist: total_dist is a float share that the observer will fill with how far it thinks ROMI has travelled in mm.
