\doxysubsection{Detailed Description}
User Interface. 

The UI task waits for the user to start the step response tests. It sets flags to turn the motors on and off and to have the data collector start saving data. It will also auto iterate through all tests needed. Once the run is complete, data for position and velocity step responses at 10 different speeds will be printed for both motors.
