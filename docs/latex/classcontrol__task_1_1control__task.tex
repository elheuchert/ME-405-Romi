\doxysubsection{Detailed Description}
This file contains a class to create a \doxylink{classcontrol__task_1_1control__task}{control\+\_\+task} object which is compatible with the cotask scheduler to run as a cooperative task. 

The active \doxylink{classcontrol__task_1_1control__task}{control\+\_\+task} object uses a finite state machine design to control the motion of the Pololu ROMI kit used in ME 405, Fall 2025. The task is given 4 PI controllers made in main that are linked to the left wheel, right wheel, the centroid of the line sensor, and the heading of the IMU sensor. Each state sets the appropriate set point of each controller according the expected motion of the bot.
